# [PromptSkiller] 路线图

本文档按“当前进度 + 下一步”管理，不再保留过期计划文本。

## 里程碑 0：工程初始化（已完成）

- Next.js + Tailwind + TypeScript 脚手架完成。
- Supabase 连接与基础数据库迁移完成。
- 基础质量命令可运行（lint/typecheck/test/build）。

## 里程碑 1：训练闭环 MVP（已完成）

- 登录/注册流程可用。
- 题库页上线（列表 + 搜索）。
- 今日训练上线（每日推荐 3 题）。
- 训练提交 -> 教练反馈 -> 历史记录闭环跑通。
- 支持本地记录与登录后云端记录。

## 里程碑 2：周赛 MVP（已完成）

- 周赛列表页与详情页可用。
- 作品提交（artifact + prompt log）可用。
- 点赞投票与取消投票可用。
- 独立作品分享页可用。
- 个人页可看到自己的周赛提交与获赞统计。

## 里程碑 3：MVP+1 训练升级（已完成）

- 三题型并行：`prompt_case` / `code_case_multi` / `build_sim_case`。
- 复杂题附件体系：`drill_assets` + 训练页附件 Tab。
- 会话轮次模型：`drill_sessions` / `drill_session_rounds`。
- 反馈模式：`过程引导` / `终局评分`。
- `build_sim_case` 结构化产物展示（summary/changed_files/patch/risk）。
- 后台题型管理与附件 CRUD。

## 里程碑 4：模块体系最小版（已完成）

- 新增 `drill_modules` / `drill_module_items`。
- 题库新增“题库视图 / 模块视图”切换。
- 支持模块过滤、模块题目序列浏览。
- 支持模块进度显示（登录态云端聚合 + 未登录本地回退）。

## 里程碑 5：MVP 收口与造数（已完成）

- 新增 `drill_user_progress` 聚合表与触发器。
- 反馈模式默认值可配置（`/settings`）。
- 终局模式支持中途简版评分（不泄露改进建议）。
- 评分维度扩展到 6 维（含 `process_control`）。
- 题库扩充到 14 题，排期扩充到 42 条，后台可直接验收示例统计。

## 里程碑 6：教学样板与内容运营规范（已完成）

- 新增 `template_case` 题型（只读看板，不参与答题提交流程）。
- 新增 `drill_template_rounds`，支持固定 2-3 轮样板提示词展示。
- 新增每日内容发布规范文档与项目内 skill。
- 新增每日 seed 生成脚本，统一 SQL 命名与批次规范。

## 里程碑 7：上线准备（进行中）

- 文档与项目状态同步（持续更新）。
- 待补：部署文档与发布 checklist。
- 待补：运营数据看板（留存、训练迭代、模块完成率、周赛参与）。
- 待补：异常监控与日志聚合。

## 里程碑 8：质量与增长（待开始）

- AI 调用限流、配额与成本控制。
- `final_only` 模式可用性验证与 A/B。
- 周赛复盘机制（优秀案例归档）。

## 里程碑 9：高阶能力探索（待开始）

- 真实沙箱执行可行性评估（`build_sim_case` 进阶）。
- 多端体验（移动端包壳）评估。
- 推送与打卡提醒策略。
- 订阅/高级功能模型验证（如有需求再开）。
