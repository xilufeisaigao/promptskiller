# [PromptSkiller] 路线图

这是一个务实的交付计划，保证首发足够小且可验证。

## 里程碑 0：项目初始化（已完成 / 进行中）

- Next.js + Tailwind
- shadcn/ui 初始化配置
- Supabase 项目创建

## 里程碑 1：MVP 核心（Web）

- Auth（Supabase）
- 每日训练：
  - 展示今日训练题
  - 输入提示词
  - 教练反馈
  - 提交历史
- 基础个人主页：
  - 训练完成次数
  - 连续打卡（简单版）

## 里程碑 2：周赛

- 挑战列表 + 详情页
- 提交流程
- 投票 + 简单榜单

## 里程碑 3：内容运营与质量

- 简单内容管理流程（seed 脚本或最小 admin UI）
- 优化教练 rubric 与 UI 渲染
- 基础反滥用（限流、举报）

## 里程碑 4：移动端打包（Capacitor）

- Web App 包壳
- 每日训练推送通知
- 可选：离线缓存与更快加载
