# [PromptSkiller] 路线图

本文档按“当前进度 + 下一步”管理，不再保留过期计划文本。

## 里程碑 0：工程初始化（已完成）

- Next.js + Tailwind + TypeScript 脚手架完成。
- Supabase 连接与基础数据库迁移完成。
- 基础质量命令可运行（lint/typecheck/test/build）。

## 里程碑 1：训练闭环 MVP（已完成）

- 登录/注册流程可用。
- 题库页上线（列表 + 搜索）。
- 今日训练上线（每日推荐 3 题）。
- 训练提交 -> 教练反馈 -> 历史记录闭环跑通。
- 训练页可调分栏（左右/上下拖拽），适配桌面端密集使用。
- 支持本地记录与登录后云端记录。

## 里程碑 2：周赛 MVP（已完成）

- 周赛列表页与详情页可用。
- 作品提交（artifact + prompt log）可用。
- 点赞投票与取消投票可用。
- 独立作品分享页可用。
- 个人页可看到自己的周赛提交与获赞统计。

## 里程碑 3：上线准备（进行中）

- 文档与项目状态同步（本次已完成一轮清理与快照）。
- 待补：部署文档与发布 checklist。
- 待补：运营数据看板（留存、训练迭代、周赛参与）。
- 待补：异常监控与日志聚合。

## 里程碑 4：质量与增长（待开始）

- 题库内容管理工具（最小 admin）。

- 周赛复盘机制（优秀案例归档）。

## 里程碑 5：多端与商业化探索（待开始）

- 移动端包壳（Capacitor）可行性评估。
