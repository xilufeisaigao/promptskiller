# [PromptSkiller] 验收清单（当前版本）

本文档用于快速验证当前产品闭环是否正常（截至 2026-03-01）。

## 0. 先决条件

- 已安装 Node.js + npm。
- 已安装 `psql`（用于 `db:*` 脚本）。
- 项目根目录存在 `.env.local`（不要提交到仓库）。
- `.env.local` 至少包含：
- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `SUPABASE_DB_URL`

## 1. 基础健康检查

在项目根目录执行：

```bash
npm run db:migrate
npm run db:seed
npm run db:status
npm run lint
npm run typecheck
npm test
npm run build
```

预期结果：

- `db:migrate` 成功执行到 `009_template_case_rounds.sql`。
- `db:status` 可连通数据库并输出模块相关表：
- `drill_assets`
- `drill_sessions`
- `drill_session_rounds`
- `drill_modules`
- `drill_module_items`
- `drill_user_progress`
- `drill_template_rounds`
- `lint/typecheck/test/build` 全通过。

## 2. 本地启动

```bash
npm run dev
```

浏览器打开 `http://localhost:3000`。

## 3. 手工验收路径

### 3.1 首页与导航

- 首页能正常打开。
- 顶部导航可进入：`今日训练`、`题库`、`周赛`、`配置`。
- 首页模块入口文案显示“已上线 Beta”。
- 管理员账号登录后，顶部额外出现 `后台` 入口。

### 3.2 题库页（双视图）

路径：`/drills`

- 默认进入“题库视图”，可搜索 + 标签筛选 + 排序。
- 可切换“模块视图”，看到模块卡片、进度条、模块内题目序列。
- 登录状态下提交训练后，模块/题库中的练习次数会实时刷新（云端聚合）。
- 支持 `?view=modules` 直达模块视图。
- 支持 `?module=<slug>` 在题库视图中过滤该模块题目。
- 卡片显示题型标签（普通题/多文件题/模拟构建题/教学样板题）。

### 3.3 今日训练（核心）

路径：`/drills/today`

- 无排期时：显示每日推荐 3 题（UTC 日期文案可见）。
- 有排期时：文案提示“管理员已发布今日题单”。
- 排期不足 3 题时，自动补齐题卡。
- 点击题卡可切换训练题。

附件与题型展示：

- 训练页左侧可切换 `题面/文件/日志`。
- 普通题无附件时不报错。
- 复杂题可查看多文件与日志内容。
- `template_case`（教学样板题）进入只读看板，不出现提交输入框。

反馈模式：

- 登录后可切换 `过程引导` / `终局评分`。
- `过程引导`：每轮提交后立即显示评分。
- `终局评分`：每轮提交后先隐藏评分，点击“完成会话并生成终局评分”后统一显示。
- 若在 `/settings` 开启“终局中途查看”，可手动查看当前轮次简版评分（仅分数，无改进建议）。
- 评分维度包含 6 项：上下文、约束、输出格式、验收标准、测试/边界、流程控制（总分 120）。

`build_sim_case`：

- 每轮提交后出现“本轮模拟产物”卡片（summary/changed_files/patch_preview）。

### 3.4 配置在线模型（可选）

路径：`/settings`

- 服务商下拉包含 `OpenAI`、`阿里云百炼（兼容模式）`、`自定义（OpenAI 兼容）`。
- 填入 key 后“测试并保存”成功时显示连接正常。
- 失败时显示错误，且不保存错误配置。

### 3.5 登录与云端存档

路径：`/auth`

- 注册/登录可用（是否邮件验证由 Supabase 配置决定）。
- 登录后提交训练题，刷新后可看到云端历史。
- 后台可查看该题提交统计。

### 3.6 周赛提交与投票

路径：`/challenges` -> `/challenges/[slug]`

- 能看到当前周赛并进入详情页。
- 登录后可提交作品（`artifact_url` + `prompt_log`）。
- 提交后列表可见，且可打开 `/submissions/[id]` 分享页。
- 点赞按钮可切换状态，且不能给自己投票。

### 3.7 管理后台（管理员）

路径：`/admin`

- 非登录用户访问提示先登录。
- 普通用户访问提示无权限。
- 管理员可：
- 新增/编辑题目（含题型）
- 发布/下线题目
- 维护每日排期（含批量）
- 维护题目附件（新增/编辑/删除）
- 查看单题提交统计
- 可直接看到示例数据（若已执行 `009_module_schedule_and_demo_data.sql`）

## 4. 边界与默认行为

- 数据库读取失败时，题库自动回退本地题目，不显示空白页。
- 未登录训练默认本地存储；登录后写入云端会话与轮次。
- 终局评分模式下，轮次评估先存档后展示，不提前泄露。
- 关键写入表均受 RLS 保护。
